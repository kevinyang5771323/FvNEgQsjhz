# 旅游景点推荐系统 python836

## 项目概述

本项目是基于 Python 开发的旅游景点推荐系统，通过分析用户喜好和行为数据，为用户提供个性化的旅游推荐。

## 技术栈

- **后端：** Python，使用 Flask 框架
- **数据库：** MySQL
- **前端：** HTML/CSS/JavaScript，Vue.js

## 功能模块

- 用户管理
  - 用户注册与登录
  - 用户信息编辑
- 旅游景点数据管理
  - 旅游景点数据录入
  - 旅游景点数据更新
- 推荐系统
  - 用户行为分析
  - 个性化推荐

## 系统角色

- 普通用户：注册、登录、查看推荐景点
- 管理员：管理旅游景点数据、用户数据

## 运行环境

- 操作系统：Linux 或 macOS
- Python 版本：3.6+
- 数据库：MySQL 5.7+

## 部署步骤

1. 安装 Python 和 Flask
2. 配置 MySQL 数据库
3. 克隆项目代码到本地
4. 运行数据库迁移脚本
5. 启动 Flask 应用

## 核心亮点

- **个性化推荐：** 结合用户历史行为，提供更精准的旅游推荐
- **易用性：** 界面简洁，操作方便，易于上手
- **可扩展性：** 模块化设计，方便后续扩展和优化系统功能
